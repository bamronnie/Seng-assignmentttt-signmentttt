Intro to CSS:

We started by identifying what CSS was, its a W3C standard for describing the apperance of HTML elements. CSS is like the decoration of your webpage, it beautifies the webpage.

After that we look at the syntax of the CSS. Here we found out that a CSS document consists of one or more style rules. And that a rule consists of a selector that identifies the Html element/s that will be affected, then followed by a series of Property-value pairs(each pair is called a declaration).

Folowing that was the locations of the CSS code where we learnt that there are three locations: inline, Internal and External.

  • Inline: Here the CSS is placed within an Html element via the style attribute.
  • Internal(Embedded): Here the CSS is placed within the style element that is inside the head element of the HTML document
  • External: Here the style rules are placed within an eternal text file with the .css extention.

How web Works:

We disccused internet Protocols in this topic, we didiused that internet Protocols have been implemented in every opertating system, and make fast web development possible.

There are MANYapplication layer protocols. web developers should be aware of the following:

  • HTTP: The hypertext Transfer Protocol is used for web communications.
  • SSH: The secure Shell Protocol Allows remote command-line connection to servers.
  • FTP: The File Transfer Protocol is used for transferring files between computers.
  • POP/IMAP/SMTP: Email-related protocols for storing and transferring emails.
  • DNS:The Domain Name System Protocol used for resolving domain names to ip address.
